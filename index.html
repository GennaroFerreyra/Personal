¿De qué trata Diseño y Desarollo Web?

La materia Diseño y Desarrollo Web se enfoca en la creación de sitios y aplicaciones web, combinando tanto aspectos visuales como funcionales. Incluye los siguientes temas:

Diseño Web (Front-End)

  • HTML.
  • CSS.
  • JavaScript.

Desarrollo Web (Back-End)

  • Lenguajes de programación: PHP, Python, JavaScript, Ruby, Java, etc.

Otras Tecnologías

  • Versionado de código: Git y GitHub.
